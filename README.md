# something-old

대학교 졸업작품으로 만든 학교내 소개팅 어플입니다.

大学卒業作品で作った学校内出会いアプリです。

## Demo / 데모 / デモサイト

[Link Open](https://something-old.netlify.app/#/)

## Getting Started / 어떻게 시작하나요? / 始め方

### Installing / 설치 / インストール

```
npm install
```

### Run / 실행 / 実行

```
quasar dev
```
